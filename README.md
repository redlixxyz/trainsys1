# Trainsys1 - Local Train Dashboard

Run locally:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Open http://127.0.0.1:5000 in your browser.

This app provides a simple embedded-style train dashboard: configure wagons, login, and view a dynamic train overview with doors and hydraulic pressure bars.
